# IHStudy Platform

IHStudy Platform is a cross-platform educational platform built using React Native, Tailwind CSS, and Next.js. This project aims to provide an integrated learning experience for students and educators.

## Features

- Interactive learning modules
- User authentication and registration
- Real-time collaboration for study groups
- Progress tracking and analytics
- Responsive design for mobile and web

## Technologies

- React Native
- Tailwind CSS
- Next.js
- Firebase (for authentication and real-time features)
